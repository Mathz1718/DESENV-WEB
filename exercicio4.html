Pergunta

Que significa semântica? Por que isso é tão importante em HTML5?

Resposta

Semântica se refere ao significado das palavras e, no contexto da web, significa usar as tags HTML apropriadas para descrever corretamente o conteúdo. Por exemplo, usar <header> para cabeçalhos, <article> para artigos, <footer> para rodapés, entre outros elementos que dão sentido ao conteúdo da página.

Ou seja, usar HTML semântico ajuda tanto os desenvolvedores quanto os usuários, tornando a web mais acessível, estruturada e eficiente.
